# Students-Result

## Overview
This project is a console-based Java application that processes
student results using control flow statements.

## Features
- Accepts student marks as input
- Validates marks (0â€“100)
- Calculates total and percentage
- Assigns grades using if-else
- Allows multiple student entries using loops
- Uses break and continue statements

## Technologies Used
- Java

## working principle 
1. User enters student name
2. User enters marks for 5 subjects
3. Program validates each mark
4. Percentage is calculated
5. Grade is assigned
6. Result is displayed
